# Algorithms

Collection of fundamental algorithms and data structures implemented in Python.

## Included Topics

### Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort

### Number Theory

* Euclidean Algorithm (GCD)
* Least Common Multiple (LCM)
* Sieve of Eratosthenes
* Prime Number Algorithms

### Graph Algorithms

* Depth-First Search (DFS)
* Breadth-First Search (BFS)
* Dijkstra's Shortest Path Algorithm

### Combinatorial Optimization

* Traveling Salesman Problem (TSP) solved using permutations (brute-force approach)
